## 🇧🇷 Português

Sistema de Reabilitação Fisioterapêutica Assistido por Inteligência Artificial
Visão Geral

Este projeto foi desenvolvido com o objetivo de auxiliar processos de reabilitação fisioterapêutica por meio de técnicas de Inteligência Artificial e Visão Computacional.

Utilizando estimativa de pose humana em tempo real, o sistema captura e analisa os movimentos realizados pelo fisioterapeuta e pelo paciente, comparando posições corporais e alinhamentos articulares para avaliar a qualidade da execução dos exercícios.

A aplicação fornece feedback visual ao usuário, classificando os movimentos como alinhados, parcialmente alinhados ou desalinhados, contribuindo para uma execução mais precisa dos exercícios e auxiliando o acompanhamento do processo de recuperação motora.

Funcionalidades
Detecção de pose humana em tempo real;
Comparação entre movimento de referência e movimento executado;
Avaliação do alinhamento corporal;
Feedback visual instantâneo;
Suporte a exercícios de fisioterapia e reabilitação motora.
Tecnologias Utilizadas
TensorFlow.js
PoseNet
JavaScript
HTML5
CSS3
WebRTC
Requisitos

Atualmente, o projeto deve ser executado por meio de um servidor web local, pois os navegadores modernos impõem restrições de segurança ao acesso da câmera e ao carregamento de modelos de aprendizado de máquina quando arquivos HTML são abertos diretamente.

Pré-requisitos
XAMPP instalado;
Navegador moderno (Google Chrome, Microsoft Edge ou Mozilla Firefox);
Webcam conectada ao computador.
Como Executar o Projeto
Instale o XAMPP em seu computador;
Copie a pasta do projeto para o diretório htdocs;
Inicie o serviço Apache pelo Painel de Controle do XAMPP;
Abra o navegador e acesse:
http://localhost/nome-da-pasta-do-projeto/index.html
Autorize o acesso à câmera quando solicitado.
Melhorias Futuras
Monitoramento remoto de pacientes;
Histórico de exercícios realizados;
Relatórios de desempenho;
Suporte a novos exercícios de reabilitação;
Aprimoramento das métricas de similaridade entre movimentos.
Contexto Acadêmico

Este projeto foi desenvolvido no contexto de uma Iniciação Científica voltada à aplicação de Inteligência Artificial e Visão Computacional na área da fisioterapia, com foco no auxílio à recuperação motora e no acompanhamento da execução de exercícios terapêuticos.



## 🇺🇸 English

AI-Assisted Physiotherapy Exercise Monitoring
Overview

This project was developed to support physiotherapy rehabilitation through Artificial Intelligence and Computer Vision techniques.

Using real-time human pose estimation, the system captures and analyzes the movements performed by both the physiotherapist and the patient, comparing body positions and joint alignments to evaluate exercise execution quality.

The application provides visual feedback by classifying movements as aligned, partially aligned, or misaligned, helping patients perform exercises more accurately and assisting physiotherapists in monitoring rehabilitation progress.

Features
Real-time human pose detection
Comparison between reference and patient movements
Body alignment assessment
Instant visual feedback
Support for physiotherapy and motor rehabilitation exercises
Technologies Used
TensorFlow.js
PoseNet
JavaScript
HTML5
CSS3
WebRTC
Requirements

Currently, the project must be executed through a local web server because browser security policies restrict direct access to camera resources and machine learning models when opening HTML files directly.

Prerequisites
XAMPP installed
Modern web browser (Google Chrome, Microsoft Edge, or Firefox)
Webcam
Running the Project
Install XAMPP.
Copy the project folder into the htdocs directory.
Start the Apache service through the XAMPP Control Panel.
Open your browser and access:
http://localhost/project-folder/index.html
Allow camera access when prompted.
Future Improvements
Remote physiotherapy monitoring
Exercise history tracking
Performance reports
Support for additional rehabilitation exercises
Improved movement similarity metrics
Academic Context

This project was developed as part of an undergraduate research initiative focused on applying Artificial Intelligence and Computer Vision to physiotherapy rehabilitation and motor recovery support.
